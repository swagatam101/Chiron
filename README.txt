===========
Chiron
===========
Chiron is a set of packages containing several analysis and 
visualization tools/algorithms for Chromatin Conformation
Capture data. 

Chiron stands for "CHromatin Informatics, Regional Organization
and Network"

In the current release, only topological analysis suite
is available. 

Read the docs. 

Typical usage: 
	import Chiron 

All the functions are available in chiron namespace.
